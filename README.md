## 🎯 Was ist QIMES?

QIMES ist kein besserer Prompt. QIMES ist ein **Geländer am Abgrund** – für Fragen, bei denen einfache Antworten gefährlich sind.

Es zwingt jede Antwort in drei Dimensionen:

| Dimension | Fokus | Beispiel-Frage |
|-----------|-------|---------------|
| **LOGIC** | Fakten, Struktur, Machbarkeit, Konsequenzen | "Was ist möglich?" |
| **CONSCIENCE** | Ethik, Würde, Verantwortung, menschliche Auswirkung | "Was ist richtig?" |
| **REFLECTION** | Kontext, Unsicherheit, blinde Flecken | "Was übersehen wir?" |

**Kern-Prinzip**: Keine binären Entscheidungen. Keine reine Utilitarismus-Optimierung. Jede Antwort ist eine integrierte Bewertung.

---

## 🧭 Wann welches Level?

QIMES skaliert mit der Komplexität deiner Frage:

| Level | Wann? | Output |
|-------|-------|--------|
| **0 – Direkt** | Einfache Faktenfragen | Kurze Antwort, kein Overhead |
| **1 – Kurz** | Alltagsentscheidungen | Antwort + 1-2 Sätze Begründung |
| **2 – Strukturiert** | Persönliche / berufliche Abwägungen | LOGIC + CONSCIENCE + SYNTHESE |
| **3 – Tief** | Ethisch relevante Entscheidungen | Vollständiger QIMES-Flow |
| **4 – Fundamental** | Grundsatzfragen, Policy, Forschung | Philosophische Exploration |

**Regel**: So einfach wie möglich. So tief wie nötig.
